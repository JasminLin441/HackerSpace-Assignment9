This repository contains the implementation for the assignment on linear cryptanalysis.

---

## Objectives
The main objectives of this project are following:
- Generate and interpret **Linear Approximation Tables**
- Identify useful linear approximation with high bias
- Perform a key recovery experiment using linear cryptanalysis
- Compare the DES and AES S-boxes

---

## Project Structure
- task1.py # LAT generation
- task2.py # Key recovery experiment using linear approximations
- task3.py # LAT analysis for DES and AES S-boxes
